# A create modpack with many addons, automate everything!

**A lightweight modpack with only 70+ mods, most being create addons.**

Despite being light, the pack includes many popular Create addons that expand storage, transportation, decoration, and automation while maintaining a mostly vanilla Minecraft experience.

**Features**
- Create and addons
- Physics contraptions with Create Aeronautics
- More recipes for automation with Ultimate factory
- Expanded storage and logistics
- Factory building and automation
- Better terrain generation with Tectonic

And other mods that make vanilla more interesting, such as:

**Better Than Vanilla**
- Vercte's Satchels
- Sophisticated backpacks 
- Tom's storage 
- Intentional access to the nether roof
- Farmer's Delight
- Easy villagers 
- And more!

## Performance Tips

**If you're playing on lower-end hardware, consider the following:**

- Reduce simulation distance to 5 chunks.
- Allocate 6–8 GB of RAM to Minecraft.
- Pre-generate chunks with Chunky (choose radius).
  - /chunky radius 2000
  - /chunky start

**Optional mods that can be removed for additional performance:**

- Create Aeronautics
- Sable
- Sable Create Addition Compat
- Sable Stuff & Additions Compatibility
- Tectonic
